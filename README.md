# Financial Transaction Categorization using TFHE library 
This program uses TFHE Rust library to categorize financial transaction while the data is encrypted. The program reads transaction descriptions for specific keywords in a predefined list and assigns categories to transactions if a specific pattern is found in the descritption

### This program uses TFHE.rs which a Rust implemenation of the Fast Fully Homomorphic Encryption over the Torus scheme
